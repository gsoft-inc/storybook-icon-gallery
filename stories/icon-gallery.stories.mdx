# Icon Gallery

An icon gallery for Storybook docs that support multiple icon variants.

## Props

<Props
    components={{
        "IconGallery": IconGallery,
        "IconGallery.Item": IconGallery.Item,
        "IconGallery.Variant": IconGallery.Variant
    }}
/>

## Usage

export function renderItem(name, Icon24, Icon32) {
    return (
        <IconGallery.Item name={name}>
            <IconGallery.Variant size={24}>
                {Icon24 && <Icon24 />}
            </IconGallery.Variant>
            <IconGallery.Variant size={32}>
                {Icon32 && <Icon32 />}
            </IconGallery.Variant>
        </IconGallery.Item>
    );
}

### Copy value

By default, the copy value of a variant is the concatenation of the item name and the variant size. A custom copy value can be specified with the `copyValue` prop of the `<IconGallery.Variant />` component.

### Render function

The icon gallery have been design for flexibility at the expense of redundacy. To stop repeating over and over the same variants, you can define a custom render function.

```jsx
function renderItem(name, Icon24, Icon32) {
    return (
        <IconGallery.Item name={name}>
            <IconGallery.Variant size={24}>
                {Icon24 && <Icon24 />}
            </IconGallery.Variant>
            <IconGallery.Variant size={32}>
                {Icon32 && <Icon32 />}
            </IconGallery.Variant>
        </IconGallery.Item>
    );
}
```
